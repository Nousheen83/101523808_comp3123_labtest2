 # Weather App - COMP3123 Lab Test 2

## Student ID: 101010101

A React weather application that fetches live weather data using OpenWeatherMap API.

### Features
- Search weather by city  
- Displays temperature, icon, humidity, wind, etc.  
- Uses props, state, hooks  
- Fully responsive UI  
- Hosted on Vercel  

### API Used
OpenWeatherMap Current Weather API  
https://api.openweathermap.org/data/2.5/weather?q=Toronto&appid=34679cfd53ef6b7ad753c1a89bd516de&units=metric

### Tech Stack
React  
Axios  
Styled Components
